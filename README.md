# Field Tracker — Project 26754-0000

Keywest Asphalt Ltd. field data collection app for Hwy 1/97 Venables Valley paving project.

## Features
- Milling session tracking with stake width readings
- Paving session with truck log and OCR ticket scanning  
- Two-way sync with Google Sheets
- Application rate reconstruction engine
- End-of-day report generation
